## Quick Install

Get up and running in seconds with our one-command installer.

<Tabs>
  <TabItem label="Unix (macOS & Linux)">
    ```bash
    curl -fsSL https://gatelet.dev/install.sh | sh
    ```
  </TabItem>
  <TabItem label="Windows (PowerShell)">
    ```powershell
    powershell -ExecutionPolicy ByPass -Command "iex (iwr -UseBasicParsing https://gatelet.dev/install.ps1)"
    ```
  </TabItem>
  <TabItem label="Manual (NPM)">
    ```bash
    npm install -g gatelet
    ```
  </TabItem>
</Tabs>
